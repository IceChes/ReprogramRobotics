# Cruciatus
A robot named for it's difficulty to build and the amount of problems I had to hunt down. 

Plastic antweight.

This robot failed prematurely at its only event due to a broken brushed drive motor sparking like crazy and jamming the radio.

Designed to use 2 TT motors and a 2207 brushless motor, as well as the 0.5x2" foam tires from Fingertech. All other parts are your choice.

Ideally do not build this robot.
