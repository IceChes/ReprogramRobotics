# Kllip
KLLIP is a 4-pin connector standard that uses Stemma QT cables to connect between robot parts using 2 way serial. All Kllip modules run at 3.3V. The Kllip project is in very early development.

# SDMD Kllip Details
- attiny84 MCU
- A4950E motor drivers
- AMS1117-3v3 BEC
- 100uF input capacitor
