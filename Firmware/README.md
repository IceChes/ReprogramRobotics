# FW
Unless otherwise specified, all firmware is to be flashed via Arduino.

- conn_test - Robin2's NRF24L01 internal connection test from the Arduino forum. For use with RF nanos.
- robot_fw - Confusingly named firmware for use with robots based on RF nano and SDMD.
- sdmd - SDMD firmware. To be flashed using Minicore with the M328 board over the ICSP.
- sdmd_testbed - For checking parts of the SDMD.
- simplerx - Robin2's simpleRX NRF test.
- simpletx - Robin2's simpleTX NRF test.
- transmitter_gen_2 - FW for my unreleased gen2 transmitter.
- universal_robot_firmware - FW for robots using RF nano and conventional RC parts.
- universal_transmitter_firmware - FW for the gen1 transmitter.
